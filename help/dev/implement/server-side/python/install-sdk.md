# Instalar o Python SDK

O Python SDK é distribuído por [pypi](https://pypi.org/project/target-python-sdk). Para começar, adicione-a como uma dependência instalando-a via pip:

## Instalar

```python {line-numbers="true"}
pip install target-python-sdk
```

O código aberto pode ser encontrado em [https://github.com/adobe/target-python-sdk](https://github.com/adobe/target-python-sdk)
